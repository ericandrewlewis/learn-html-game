HTML elements

HTML is the language web pages are written in. HTML is made of elements, which each represent one item on a web page.

An HTML element looks like this:

<road></road>

The opening tag <road> consists of angle brackets with the element's name inside it.

The closing tag </road> is like an opening tag with a forward slash before the name.
